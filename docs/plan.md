# Nano Info Prompt 开发计划大纲

## 阶段一：项目初始化与基础架构

1. **项目搭建**
   - 使用 Vite + React 18 + TypeScript 初始化项目
   - 配置 Tailwind CSS
   - 安装并配置 Shadcn UI 组件库
   - 配置 Zustand 状态管理

2. **项目结构规划**
   - 创建目录结构：`components/`、`stores/`、`utils/`、`types/`、`styles/`
   - 定义 TypeScript 类型接口

---

## 阶段二：核心功能模块开发

3. **输入区模块**
   - Tab 切换组件（粘贴文本 / 上传文件 / 简单指令）
   - 文本输入框组件
   - 文件上传组件（支持 PDF、Word、Markdown、TXT）
   - 生成数量选择器（N 值选择）

4. **文件解析模块**
   - 集成 PDF.js 解析 PDF 文件
   - 集成 Mammoth.js 解析 Word 文件
   - Markdown/TXT 文本提取

5. **AI 工作流模块**
   - API 配置组件（Base URL / API Key 设置）
   - AI 请求封装（OpenAI 兼容接口）
   - 内容处理 System Prompt 集成
   - 响应解析与错误处理

---

## 阶段三：内容展示与编辑模块

6. **内容块编辑区**
   - 卡片式内容块展示组件
   - 内容块可编辑功能
   - 加载状态与骨架屏

7. **视觉风格选择模块**
   - 风格库数据定义
   - 风格标签选择组件
   - 风格模板管理

---

## 阶段四：提示词生成与输出

8. **提示词动态拼接**
   - 模板拼接逻辑（前端本地处理）
   - 内容块 + 风格模板 → Prompt 生成

9. **结果展示区**
   - Prompt 卡片流展示
   - 一键复制功能
   - 复制成功反馈

---

## 阶段五：状态管理与全局逻辑

10. **Zustand Store 设计**
    - 输入状态管理
    - 内容块状态管理
    - 风格选择状态
    - API 配置状态

11. **容错与异常处理**
    - JSON 解析失败处理
    - 数组长度不匹配提示
    - 网络错误处理

---

## 阶段六：UI 优化与部署

12. **UI 风格优化**
    - 模仿 Anthropic 视觉设计规范
    - 响应式布局适配
    - 交互动效优化

13. **部署上线**
    - Cloudflare Pages 配置
    - 环境变量配置
    - 构建与发布

---

## 开发顺序

```
阶段一 → 阶段二(3→4→5) → 阶段三(6→7) → 阶段四(8→9) → 阶段五 → 阶段六
```
